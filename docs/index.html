macOS menu bar app

Bob watches your window and jumps in.

Shake a window and Bob appears beside it, understands the app context, and starts a focused agent without turning your desktop into a giant chat app.

Getting started

  1. Download the latest build from GitHub Releases.
  2. Move Bob to Applications.
  3. Open Bob from the menu bar.
  4. Grant permissions when macOS asks.
  5. Open Finder and shake a window to try it.
