Running

Summarize this folder

Bob reads the active Finder window and starts the matching rule.

1. Shake

Move a window with intent and Bob catches the gesture.

2. Understand

Bob identifies the app and pulls the context it can extract.

3. Run

Rules launch an agent and progress appears in a floating card stack.

Getting started

Install Bob in under a minute.

  1. Download the latest build from GitHub Releases.
  2. Move Bob to Applications.
  3. Open Bob from the menu bar.
  4. Grant the macOS permissions it asks for.
  5. Open Finder and shake a window to try your first task.
