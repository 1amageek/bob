What Bob does

  • Shake a Finder window and Bob pops up right beside it
  • Bob sees which folder you are looking at and understands the context
  • Ask anything — Bob runs an AI agent to get it done
  • Watch progress live in a floating card while you keep working

What works today

  • Works with any Finder window — Bob knows the folder you are in
  • Drag files onto Bob to kick off a task instantly
  • Set up folder watch rules to automate recurring work
  • Lives quietly in the menu bar — no Dock clutter

Extend Bob

Skills

Skills are installable prompt & tool packages that give Bob new abilities. Browse curated skills or create your own to automate any workflow.

Share skills with your team by dragging the .skill file — just drop it onto Bob to install.

Get started in 3 minutes

  1. Install Claude Code — Bob uses it under the hood.
  2. Download Bob and drop it into Applications.
  3. Open Bob and allow the permissions macOS asks for.
  4. Open a Finder window and give it a shake.
  5. Say hi to Bob. He is ready to help.
