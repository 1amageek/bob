What makes Bob different

Built around the window you are using right now.

Window-aware

Bob identifies the frontmost window, the app behind it, and the context it can extract.

Rule-first

Triggers, conditions, prompts, and outputs are defined as rules instead of ad hoc actions.

Adapter-based

Finder support comes first. Other apps can be added through new adapters over time.

Non-intrusive

The floating panel stays beside your work instead of replacing it with a full-screen AI surface.

How it works

From window motion to task cards.

  1. Shake a window Bob watches for a deliberate shake gesture while you drag a window.
  2. Capture context Bob records the frontmost app, frame, and process information.
  3. Resolve an adapter The matching adapter extracts structured context from the app.
  4. Run a rule Rules expand the prompt, choose the backend, and start work.
  5. Track progress Cards show scanning, planning, editing, review, and completion.

Getting started

Install Bob and run your first window-aware task.

  1. Download the latest build from GitHub Releases.
  2. Move Bob to the Applications folder.
  3. Open Bob from the menu bar.
  4. Grant the required macOS permissions when prompted.
  5. Open Finder, select a target folder or file, and shake the window.

Initial releases are optimized for Finder-based workflows while the adapter system expands.
