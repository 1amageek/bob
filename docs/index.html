What Bob does

  • Appears next to the window you are using
  • Reads app context through an adapter
  • Runs a rule-driven agent for the current task
  • Shows progress in a floating card stack

What works today

  • Finder-based window context
  • Folder and file aware workflows
  • Rule-first task execution
  • Floating menu bar experience on macOS

How to try it

  1. Install Claude Code.
  2. Download Bob from GitHub Releases.
  3. Move Bob to Applications and open it.
  4. Grant macOS permissions when asked.
  5. Open Finder and shake a window.
